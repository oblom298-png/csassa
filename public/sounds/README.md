# 🔊 Звуки игры

## Файлы звуков:

### hit.mp3 — звук попадания в противника
Положи файл **`hit.mp3`** сюда → `public/sounds/hit.mp3`
Играет ТОЛЬКО когда ты убиваешь врага.

### s1.mp3 — звук выстрела из пистолета
Положи файл **`s1.mp3`** сюда → `public/sounds/s1.mp3`
Играет при каждом выстреле из пистолета.
